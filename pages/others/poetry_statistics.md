## 分类说明

### 先秦（374首）
- **诗经**：306首，中国最早的诗歌总集
- **楚辞**：66首，以屈原、宋玉为代表的浪漫主义诗歌
- 作者包括：无名氏（诗经）、屈原、宋玉等

### 魏晋（30首）
- 以曹操为主的建安风骨诗歌
- 主要作者：曹操(29首)、陶渊明等

### 唐（406首）
- **唐诗**：361首，中国诗歌的黄金时代
- **乐府**：45首，继承汉乐府传统
- 主要作者：杜甫(43)、李白(37)、王维(32)、李商隐(24)、孟浩然(15)、杜牧(15)、白居易(14)等84位诗人

### 南唐（4首）
- 南唐后主李煜的词作
- 代表作：《虞美人》等

### 宋（313首）
- **宋词**：311首，词的鼎盛时期
- 主要作者：吴文英(25)、周邦彦(22)、晏几道(17)、姜夔(17)、辛弃疾(16)、柳永(14)、苏轼(13)等96位词人

### 清（262首）
- 主要为纳兰性德的词作(262首)
- 清代词坛的代表人物

### 现代（4首）
- 毛泽东诗词(4首)
- 代表作：《沁园春·雪》等

### 蒙学（32首）
- **三字经**：2个版本（新版、传统版）
- **声律启蒙**：30章（上卷15章、下卷15章）
- 作者：王应麟、车万育

## 数据特点

1. **唐诗最多**：406首，占总数的28.5%，体现了唐代诗歌的繁荣
2. **先秦经典**：374首，包含了中国文学的源头作品
3. **宋词丰富**：313首，展现了宋代词坛的多样性
4. **纳兰词集**：262首清词几乎全部为纳兰性德作品，体现了其在清代词坛的地位
5. **名家荟萃**：收录了从先秦到现代的200多位作者作品

## 使用建议

可以通过 `tags` 字段进行多维度筛选：

```javascript
// 按朝代筛选
const tangPoems = data.filter(p => p.tags.includes('唐'));
const songCi = data.filter(p => p.tags.includes('宋'));

// 按体裁筛选
const shijing = data.filter(p => p.tags.includes('诗经'));
const chuci = data.filter(p => p.tags.includes('楚辞'));

// 按主题筛选
const homesick = data.filter(p => p.tags.includes('思乡'));
const landscape = data.filter(p => p.tags.includes('山水'));

// 组合筛选
const tangLandscape = data.filter(p => 
  p.tags.includes('唐') && p.tags.includes('山水')
);
```
