📊 核心数据流

1
用户连接 → Worker 路由 → 创建 WebSocket 连接到聊天室 DO
2
发送消息 → 消息校验 → 持久化存储 → 广播到所有连接用户
3
文件上传 → Worker 处理 → 上传到 R2 → 返回公开 URL
4
AI 请求 → 外部 API 调用 → 结果处理 → 返回给用户
5
定时任务 → 数据获取 → 图表生成 → 上传 R2 → 发送到聊天室
